# 数字猜猜猜 - 外网联机部署指南

## 一键启动（推荐）

双击项目目录下的 `start-online.bat` 即可。

第一次会自动下载 cloudflared（约 15MB），之后秒启动。

---

## 手动操作步骤

### 第 1 步：安装 cloudflared

打开 **PowerShell**，运行以下命令（二选一）：

**方式 A：winget（推荐）**
```powershell
winget install Cloudflare.cloudflared
```

**方式 B：手动下载**
1. 打开 https://github.com/cloudflare/cloudflared/releases/latest
2. 下载 `cloudflared-windows-amd64.exe`
3. 重命名为 `cloudflared.exe`，放到项目目录

### 第 2 步：启动服务器

```powershell
cd 项目目录
node server.js
```

看到 `🎲 数字猜猜猜联机服务器已启动` 即可。

### 第 3 步：新开一个终端，启动隧道

```powershell
cd 项目目录
cloudflared tunnel --url http://localhost:7788
```

等几秒，你会看到类似输出：
```
+----------------------------+
| https://abc-xyz.trycloudflare.com |
+----------------------------+
```

**这个 `https://xxx.trycloudflare.com` 就是你的公网地址！**
把这个链接发给朋友就行。

### 第 4 步：分享给朋友

朋友在任意设备的浏览器打开这个链接，创建/加入房间，就能跨网联机了！

---

## 注意事项

- ⚡ **你的电脑必须开着**，服务器和 tunnel 才能运行
- 🔗 **每次重启 tunnel 地址会变**（免费版不固定域名），需要重新分享链接
- 📱 **手机也能玩**，把链接发到手机浏览器打开即可
- 🎮 **同一个链接两个浏览器标签页** = 两个玩家

---

## 常见问题

**Q: 朋友连不上？**
A: 确认你电脑上的 tunnel 窗口还开着，没有报错。检查防火墙是否放行了 cloudflared。

**Q: 想要固定地址？**
A: 免费版不支持固定地址。如果需要，可以注册 Cloudflare 账号并创建一个免费的 tunnel（需要有自己的域名）。

**Q: WebSocket 连接失败？**
A: Cloudflare 免费版完全支持 WebSocket，确认 tunnel 终端显示连接正常即可。
